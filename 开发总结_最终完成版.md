## 🚀 完整的部署方案

### 后端部署

```bash
# 使用Docker Compose
docker-compose up -d --build

# 创建超级用户
docker-compose exec web python manage.py createsuperuser
```

### 前端部署

```bash
# 安装依赖
cd frontend
npm install

# 开发模式
npm run dev

# 生产构建
npm run build
```
